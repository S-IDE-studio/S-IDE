# S-IDE Mobile

## Features

- Connect to S-IDE desktop/server instances
- Terminal access from mobile device
- QR code support for easy connection
- Dark theme support

## Installation

### iOS
1. Download from TestFlight or EAS
2. Install and open the app

### Android
1. Download APK from EAS Build page
2. Install the APK
3. Open the app

## Usage

1. Scan QR code from S-IDE desktop tunnel panel
2. Or manually enter server URL
3. Connect and start using terminals
